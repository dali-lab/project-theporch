The Porch

The Porch is a web app built for the indoor rock climbing community. Allowing climbers to interact with one another, track progress, provide feedback, and generally extend the climbing community beyond the walls of the gym. At the same time we strive to provide a valuable resource for climbing gym owners so that they may manage the actual climbing within the gym; currently, there is no established and universal system by which feedback is collected, leaving gym owners reliant on sorely inadequate tools to meet an increasingly saavy and growing population of climbers.


The Porch has teamed up with the DALI Lab

  • To Create a Native Mobile App
  • To Develop a Clean Interface for Gym Administrators
  • Building Out and Perfecting Features of Interface
